# Podcasts App
Podcasts Application by React.js & Typescript

## Features

### Listener & Host

[X] Create Account
[X] Log in
- Edit Profile
[X] Log out

### Host

[X] Create a Podcast
- Edit a Podcast
- Delete a Podcast
- See Podcast Dashboard (See Listeners and Reviews)
[X] Upload an Episode
- Edit an Episode
- Delete an Episode

### Listener

- See Podcast Categories
- Search Podcsts
- See Podcast + Episodes
- Review Podcast
- Subscribe to a Podcast
- See Subscriptions
- See Feed (Episodes of Podcasts the user is subscribed to combined)
